# 轨迹追踪模块

## 模块概述

轨迹追踪模块用于追踪人脸和车辆在指定时间范围内的活动轨迹，通过地图可视化展示设备捕捉记录。

## 模块结构

```
src/views/trackTrace/
├── README.md                    # 模块说明文档
├── faceTrace/                   # 人脸轨迹追踪
│   └── index.vue               # 人脸轨迹追踪页面
└── vehicleTrace/               # 车辆轨迹追踪
    └── index.vue               # 车辆轨迹追踪页面
```

## 功能特性

### 人脸轨迹追踪 (faceTrace)

#### 搜索条件
- **人脸图片上传**：支持拖拽和点击上传，格式支持 jpg/png
- **时间范围选择**：使用日期时间范围选择器，精确到秒
- **操作按钮**：追踪和重置按钮，左右布局

#### 地图展示
- **设备标记**：在地图上标注捕捉到人脸的设备位置
- **捕捉次数显示**：每个设备标记显示捕捉次数
- **颜色区分**：
  - 绿色：1-3次捕捉
  - 橙色：4-6次捕捉
  - 红色：7次以上捕捉
- **详情弹窗**：点击设备标记显示详细捕捉记录

#### 详情表格字段
- 捕捉时间
- 设备编号
- 捕捉照片（支持预览）
- 人脸相似度（带颜色标签）
- 设备名称
- 设备位置

### 车辆轨迹追踪 (vehicleTrace)

#### 搜索条件
- **车辆图片上传**：支持拖拽和点击上传，格式支持 jpg/png
- **时间范围选择**：使用日期时间范围选择器，精确到秒
- **操作按钮**：追踪和重置按钮，左右布局

#### 地图展示
- **设备标记**：在地图上标注捕捉到车辆的设备位置
- **捕捉次数显示**：每个设备标记显示捕捉次数
- **颜色区分**：与人脸轨迹相同的颜色规则
- **详情弹窗**：点击设备标记显示详细捕捉记录

#### 详情表格字段
- 捕捉时间
- 设备编号
- 捕捉照片（支持预览）
- 车辆相似度（带颜色标签）
- 设备名称
- 设备位置

## 技术实现

### 核心技术栈
- **Vue 3 Composition API**：现代化的组件开发方式
- **Element Plus**：UI组件库
- **高德地图 API**：地图展示和标记功能
- **AMapComponent**：项目内置的地图组件

### 地图功能
- **地图初始化**：默认中心点为佛山新城
- **标记管理**：动态添加和清除地图标记
- **自定义图标**：SVG格式的圆形标记，显示捕捉次数
- **交互事件**：标记点击事件处理

### 数据模拟
- **设备位置**：模拟佛山新城区域内的监控设备
- **捕捉记录**：随机生成时间、相似度等数据
- **图片资源**：使用Unsplash高质量图片

## 样式设计

### 布局结构
- **左右布局**：与aiSearch模块保持一致
- **响应式设计**：支持桌面端、平板端、移动端
- **栅格系统**：使用Element Plus的24栅格系统

### 样式特点
- **统一风格**：与项目整体样式保持一致
- **地图容器**：高度自适应屏幕尺寸
- **上传组件**：拖拽上传样式优化
- **空状态**：友好的空状态提示

## 使用说明

### 人脸轨迹追踪
1. 访问 `/trackTrace/faceTrace`
2. 上传人脸图片
3. 选择时间范围
4. 点击"追踪"按钮
5. 查看地图上的设备标记
6. 点击标记查看详细记录

### 车辆轨迹追踪
1. 访问 `/trackTrace/vehicleTrace`
2. 上传车辆图片
3. 选择时间范围
4. 点击"追踪"按钮
5. 查看地图上的设备标记
6. 点击标记查看详细记录

## 扩展功能

### 可扩展特性
- **轨迹连线**：连接时间序列上的设备点
- **热力图**：显示活动密集区域
- **时间轴**：按时间顺序播放轨迹
- **导出功能**：导出轨迹数据和报告
- **实时追踪**：支持实时数据更新

### 性能优化
- **标记聚合**：相近设备自动聚合显示
- **懒加载**：大量数据分批加载
- **缓存机制**：地图和数据缓存
- **防抖处理**：搜索操作防抖优化

## 注意事项

1. **地图API密钥**：需要配置有效的高德地图API密钥
2. **图片格式**：仅支持jpg和png格式
3. **时间范围**：建议选择合理的时间范围以获得最佳性能
4. **浏览器兼容性**：建议使用现代浏览器以获得最佳体验
5. **网络环境**：需要良好的网络环境加载地图和图片资源
